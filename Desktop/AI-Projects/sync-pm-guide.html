The data pipeline

Every experiment slide follows a 4-step process — no manual data entry needed.

A

Classify metrics

  • Pull all metrics from Eppo
  • Look up each metric name
  • Get desired direction (↑ or ↓)
  • Bucket: primary, guardrail, secondary
B

Structure input

  • Variants + sample sizes
  • Market + entry point
  • CUPED-adjusted values
  • Pass/fail per metric
C

Write summary

  • 1 headline sentence
  • 3-5 insight bullets
  • Under 150 words
  • Plain business language
D

Build slide

  • Talabat brand template
  • Auto-formatted results
  • Next steps from data
  • Footer links from Jira
