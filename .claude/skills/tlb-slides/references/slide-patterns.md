# Talabat Slide Patterns Reference

Detailed layout specifications and pptxgenjs code patterns for every talabat slide type.

All screenshots referenced are in `../assets/screenshots/template/`.
All brand assets are in `../assets/`.

> **Screenshot rendering note:** Screenshots were generated by LibreOffice and show approximate layout/structure. Font rendering, decorative elements, and some overlays may differ from Google Slides / PowerPoint. Trust the documented specs (font, size, color, position values) as the source of truth — use screenshots for layout inspiration, not pixel-matching.

### Presentation Setup

```javascript
const pptxgen = require("pptxgenjs");
const pptx = new pptxgen();

// Talabat dimensions
pptx.defineLayout({ name: "TALABAT", width: 10, height: 5.63 });
pptx.layout = "TALABAT";

// Brand constants
const COLORS = {
  orange: "FF5900",
  burgundy: "411517",
  cream: "F4EDE3",
  white: "FFFFFF",
  lime: "CFFF00",
  purple: "8318D8",
  lavender: "EBE8FC",
  darkGray: "343B46",
  mediumGray: "595959",
};

const FONTS = {
  primary: "Poppins",
  primaryMedium: "Poppins Medium",
  primarySemiBold: "Poppins SemiBold",
  primaryBold: "Poppins ExtraBold",
  primaryBlack: "Poppins Black",
  secondary: "DM Sans",
  secondaryBlack: "DM Sans Black",
  orgChart: "Open Sans",
};
```

### Logo Helper

```javascript
// Standard logo (top-right, content slides)
function addLogo(slide, position = "standard") {
  const positions = {
    standard: { x: 8.29, y: 0.24, w: 1.39, h: 0.46, rotate: 4.7 },
    coverTopRight: { x: 7.09, y: 0.40, w: 2.52, h: 0.84, rotate: 4.7 },
    coverBottomRight: { x: 6.63, y: 4.31, w: 2.99, h: 1.00, rotate: 0 },
    bottomLeft: { x: 0.50, y: 4.85, w: 1.39, h: 0.46, rotate: 0 },
  };
  const pos = positions[position];
  slide.addImage({
    path: "assets/logo/talabat-logo.png", // adjust path as needed
    x: pos.x, y: pos.y, w: pos.w, h: pos.h,
    rotate: pos.rotate,
  });
}
```

### Accent Bar Helper

```javascript
// Orange accent bar at top-left of content slides
function addAccentBar(slide) {
  slide.addShape(pptx.shapes.ROUNDED_RECTANGLE, {
    x: 0.42, y: 0.0, w: 1.20, h: 0.07,
    fill: { color: COLORS.orange },
    line: { color: COLORS.orange, width: 0 },
    rotate: 180,
    rectRadius: 0.02,
  });
}
```

### Standard Title Helper

```javascript
function addTitle(slide, text, options = {}) {
  slide.addText(text, {
    x: options.x || 0.36,
    y: options.y || 0.18,
    w: options.w || 9.41,
    h: options.h || 0.68,
    fontFace: FONTS.primarySemiBold,
    fontSize: options.fontSize || 25,
    color: options.color || COLORS.orange,
    align: options.align || "left",
    bold: false,
  });
}
```

## Numbered Lists

### With Large Numbers (Cream)
**Screenshot**: `numbered-list-cream.jpg`

```javascript
const slide = pptx.addSlide();
// Background: white (inherited)

const items = [
  { num: "1.", title: "First Item", body: "Description..." },
  { num: "2.", title: "Second Item", body: "Description..." },
  { num: "3.", title: "Third Item", body: "Description..." },
];

items.forEach((item, i) => {
  const y = 0.5 + i * 1.5;

  // Large number
  slide.addText(item.num, {
    x: 0.5, y: y, w: 1.5, h: 1.2,
    fontFace: FONTS.secondaryBlack, // DM Sans Black
    fontSize: 100, color: COLORS.orange,
  });

  // Title + body
  slide.addText([
    { text: item.title + "\n", options: { fontSize: 16, bold: true, color: COLORS.burgundy }},
    { text: item.body, options: { fontSize: 10, color: COLORS.burgundy }},
  ], { x: 2.5, y: y, w: 7.0, h: 1.2, fontFace: FONTS.primary });
});
```

## Agenda Slides

### Dark Theme
**Screenshot**: `agenda-dark.jpg`

```javascript
const slide = pptx.addSlide();
slide.background = { color: COLORS.burgundy };

// Add decorative images behind
// slide.addImage(...)

addTitle(slide, "Agenda", { color: COLORS.orange });

// Agenda items
const sections = [
  { title: "Introduction", items: ["Background", "Context"] },
  { title: "Main Topic", items: ["Point 1", "Point 2"] },
  { title: "Conclusion", items: ["Summary", "Next Steps"] },
];

let y = 1.2;
sections.forEach(section => {
  slide.addText(section.title, {
    x: 0.5, y: y, w: 4.0, h: 0.4,
    fontFace: FONTS.primaryMedium,
    fontSize: 16, color: COLORS.orange,
  });
  y += 0.4;
  section.items.forEach(item => {
    slide.addText(item, {
      x: 0.7, y: y, w: 3.8, h: 0.35,
      fontFace: FONTS.primaryMedium,
      fontSize: 14, color: COLORS.burgundy, // same as bg for "hidden" effect
    });
    y += 0.35;
  });
  y += 0.1;
});
```

### Icon Grid (2x3)
**Screenshot**: `special-icon-grid.jpg`

```javascript
const slide = pptx.addSlide();
slide.background = { color: COLORS.cream };

addAccentBar(slide);
addLogo(slide, "standard");
addTitle(slide, "Icon Grid Title");

const items = [
  { icon: "assets/icons/image258.png", num: "01", title: "Item One", desc: "Description..." },
  // ... 6 items total
];

items.forEach((item, i) => {
  const col = i % 3;
  const row = Math.floor(i / 3);
  const x = 0.59 + col * 3.38; // columns at 0.59, 3.97, 7.35
  const y = 1.2 + row * 2.0;

  // Icon
  slide.addImage({ path: item.icon, x: x, y: y, w: 0.5, h: 0.5 });

  // Number + Title
  slide.addText([
    { text: item.num + ". ", options: { bold: true }},
    { text: item.title },
  ], {
    x: x, y: y + 0.6, w: 2.44, h: 0.3,
    fontFace: FONTS.primarySemiBold,
    fontSize: 12, color: COLORS.burgundy,
  });

  // Description
  slide.addText(item.desc, {
    x: x, y: y + 0.95, w: 2.44, h: 0.8,
    fontFace: FONTS.primary,
    fontSize: 9, color: COLORS.burgundy,

});
```

## BFF Vision Example Patterns

Screenshots of the real-world BFF Vision 2027 presentation are in `../assets/screenshots/example-bff/`.

### Key Patterns Observed

1. **Consistent cream background** (`#F4EDE3`) for all content slides
2. **Orange accent bar** on every slide (round2SameRect at top-left)
3. **Logo always top-right** at (8.29", 0.24") rotated 4.7 degrees
4. **13pt body text** (slightly larger than template's 10pt) — Poppins Regular
5. **Complex diagrams** built from many grouped shapes (architecture flows, comparison tables)
6. **Click animations** on most slides (progressive reveal)
7. **DM Sans** used as alternate body font on one slide (slide 20)
8. **Additional diagram colors**: `#A2C4C9` (teal), `#EAD1DC` (rose), `#FFC944` (gold), `#FFD1B3` (peach)

## Reusable Template Scripts

Ready-to-run pptxgenjs scripts in this directory:

### Experiment Review v2 (`create-experiment-v2.js`)
Single-slide experiment review template with placeholder text. Two-column layout: Experiment Setup (left) + Experiment Results (right). Includes hypothesis badge, variant badges (A/B), market/entry point, mockup placeholder, results with lime-highlighted metrics, note/caveats bar, next steps, and footer with 1 Pager | Eppo | Working Sheet | Data PIC links.

### Lightspeed v2 (`create-lightspeed-v2.js`) — SUPERSEDED by v3

### Lightspeed v3 (`create-lightspeed-v3.js`)
Initiative update template with reusable helper functions. Layout matches the "Improving order tracking experience" reference slide. Features:
- **Status badge** (top-right): configurable text + color (e.g. "A Big Win!", "In Progress", "Shipped")
- **Impact headline**: Highlighted numbers with lime background
- **Left column**: What? / Impact / Next steps — structured bullet sections
- **Right column**: Two visual blocks, each with bold header, description, and screenshot placeholder
- **Helper functions**: `addBaseLayout`, `addImpactHeadline`, `addWhatSection`, `addImpactSection`, `addNextSteps`, `addRightVisuals` — all reusable for generating filled decks from Jira data

**Used by `/sync artifacts` → lightspeed** to generate initiative update slides.

### 3-Experiment Filled Deck (`create-3-experiments.js`)
Real example of a multi-slide experiment deck filled with Eppo data. Contains helper functions (`addBaseLayout`, `addColumnHeaders`, `addHypothesis`, `addVariant`, `addFooter`, `metricLine`) that are reusable for generating any experiment review deck. Key patterns:
- `metricLine()`: Formats metric with lime highlight (positive/neutral) or red highlight (`FFCDD2`) for significant negative guardrails
- 3-variant layout: Adds a third variant badge at a lower y-position with `medGray` color
- Significance indicators: ✅ for positive significant, ⚠️ for guardrail violations

**Used by `/sync experiment` to generate experiment review slides.** The workflow:
1. Pull experiment setup from Jira description + 1-pager docs
2. Pull results from Eppo MCP (`get_experiment_results`, `get_experiment_details`, `get_metric_details`)
3. Generate a pptxgenjs script using these helper functions with real data
4. Run with node to create .pptx
5. Upload to Google Drive via `create_drive_file`
