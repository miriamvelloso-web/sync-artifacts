Never write to the live Opportunity Intake Sheet (or any shared Google Sheet) until Miriam explicitly gives the go-ahead with words like "please start" or "apply". Dry run reports are fine — but writing is blocked until she confirms.

**Why:** The intake sheet is shared with stakeholders. Writing prematurely triggers the intake bot to send DMs and changes data others are working with. Miriam needs to review and approve before anything goes live.

**How to apply:** When running intake-triage or any sheet-writing automation, always stop after the dry run report and wait for explicit approval before writing. "ok" or "yes start" to building the skill does NOT mean "write to the sheet."
